# 🚀 Employee Management System (Full Stack)

## 📌 Description

A full-stack Employee Management System built using React and Spring Boot with secure JWT authentication. This application allows users to manage employee data efficiently with complete CRUD operations.

---

## ✨ Features

* 🔐 Secure Login using JWT Authentication
* 👨‍💼 Add, Update, Delete Employees
* 📊 Dashboard with employee statistics
* 🔍 Search and filter employees
* ⚡ REST API integration
* ❗ Global Exception Handling

---

## 🛠 Tech Stack

**Frontend:** React.js, JavaScript, CSS
**Backend:** Spring Boot, Java
**Database:** MySQL
**Tools:** Git, GitHub, Postman, VS Code

---

## ⚙️ Setup Instructions

### 🔹 Backend

1. Navigate to backend folder
2. Run command:
   mvn spring-boot:run
3. Runs on: http://localhost:8080

---

### 🔹 Frontend

1. Navigate to frontend folder
2. Run commands:
   npm install
   npm start
3. Runs on: http://localhost:3000
