# Employee Management System

This is a full stack web application to manage employee records.

## Tech Stack
- React.js
- Spring Boot
- MySQL

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employees
- Login Authentication
- Search, Filter and Sort

## How to Run

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm start  

## API Endpoints
- GET /employees
- POST /employees
- PUT /employees/{id}
- DELETE /employees/{id}
